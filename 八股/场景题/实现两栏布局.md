1.使用flex
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>大文件分片上传</title>
    <input type="file">
    <style>
        .container {
            display: flex;
            width: 800px;
            height: 400px;
        }

        .left {
            width: 200px;
            height: 100%;
            background-color: rgba(0, 0, 255, 0.1);
        }

        .right {
            /* overflow: auto; */
            flex: 1;
            height: 100%;
            background-color: rgba(255, 0, 0, 0.1);
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="left"></div>
        <div class="right"></div>
    </div>
</body>
</html>
```

4.使用bfc+浮动
应用原理：BFC 的区域不会和浮动区域重叠，所以就可以把侧边栏固定宽度且左浮动，而对右侧内容触发 BFC，使得它的宽度自适应该行剩余宽度。
```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>大文件分片上传</title>
    <input type="file">
    <style>
        .container {
            /* display:grid; */
            position: relative;
            width: 800px;
            height: 200px;
        }
        .left {
            float: left;
            width: 200px;
            height: 100%;
            background-color: rgba(0, 0, 255, 0.1);
        }
        .right {
            overflow: auto;
            height: 100%;
            background-color: rgba(255, 0, 0, 0.1);
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="left"></div>
        <div class="right"></div>
    </div>
</body>
</html>
```
