
对于一个大文件，我们进行上传的时候肯定不能直接上传，否则占用服务器带宽，且一旦出现网络波动，又要重新上传整个文件。

因此大部分大文件是采用的分片上传。

html
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>大文件分片上传</title>
    <input type="file">
</head>
<body>
    <script src="./fileUpload.js">
    </script>
</body>
</html>
```
